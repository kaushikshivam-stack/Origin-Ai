# ORIGIN - AI Platform

A modern, premium AI chatbot landing page built with React.js and Tailwind CSS.

## ⚙️ Tech Stack

- React.js
- Vite
- Tailwind CSS

## 🔋 Features

- **Beautiful Sections**: Hero, Services, Features, How to Use, Roadmap, Pricing, Footer, and Header
- **Parallax Animations**: Engaging effects triggered by mouse movement and scrolling
- **Modern UI Design**: Glassmorphism, gradient effects, and dark mode
- **Responsive**: Seamless functionality across all devices
- **Cool Gradients**: Stylish gradients for cards, buttons, and backgrounds

## 🤸 Quick Start

**Prerequisites**

- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/)

**Installation**

```bash
npm install
```

**Running the Project**

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the project.
